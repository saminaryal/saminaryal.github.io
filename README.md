# Samin Aryal Profile Website

A modern, responsive personal portfolio website prepared for GitHub Pages.

## Included

- Professional homepage
- About section
- Work experience
- Leadership experience
- Projects and field experience
- Skills and languages
- Education timeline
- Contact section
- Light/dark mode
- Mobile-responsive layout
- CV view button

## Publish with GitHub Pages

1. Create a public GitHub repository.
2. Upload all files and folders from this project.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Choose the `main` branch and `/ (root)`.
6. Click **Save**.
7. Wait for GitHub to publish the website.

For a personal GitHub Pages address such as:

```text
https://saminaryal.github.io
```

the repository must be named:

```text
saminaryal.github.io
```

and it must belong to the GitHub account named `saminaryal`.
