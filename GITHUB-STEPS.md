# GitHub Steps

## Create the repository

1. Sign in to GitHub.
2. Click **+ → New repository**.
3. For a direct profile link, use the repository name:
   `saminaryal.github.io`
4. Select **Public**.
5. Click **Create repository**.

## Upload the website

1. Extract the downloaded ZIP file.
2. Open the extracted folder.
3. Select everything inside it:
   - `index.html`
   - `assets`
   - `README.md`
   - `.nojekyll`
4. Drag these items into the empty GitHub repository.
5. Enter the commit message:
   `Upload personal profile website`
6. Click **Commit changes**.

## Enable GitHub Pages

1. Open **Settings**.
2. Open **Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select:
   - Branch: `main`
   - Folder: `/ (root)`
5. Click **Save**.

The website should be available at:

`https://saminaryal.github.io`

## Future editing

- Profile text: edit `index.html`
- Design: edit `assets/css/style.css`
- Photo: replace `assets/images/samin-aryal.jpg`
- CV: replace `assets/files/Samin-Aryal-CV.pdf`
